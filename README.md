# TropicalGlaciers
Depository for "Quantifying downstream environmental impacts of disappearing tropical glaciers in contrasting climates" project

This repository includes the data and python/Google Earth Engine scripts needed for this project's analysis. 

The Google Earth Engine Scripts can be used to extract CHIRPS precipitation data and MODIS NDVI data for study areas of interest. The GEE files include links to GEE projects and pasted scripts as backup.

The python notebook (TropicalGlaciers_Figures) contains cells that can perform NDVI smoothing, correlation analysis, NDVI seasonal comparison, and climate condition comparison. They can be run with the data inside the "GlacierProject" folder in this repository. You can download the folder and update "BASE_DIR" in each cell with the folder's location on your computer.
